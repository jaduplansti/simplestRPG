# simplestRpg
A turn based rpg game built with sticks and stones.

# Installation
```
git clone https://github.com/jaduplansti/simplestRPG
cd simplestRpg
pip install -r requirements.txt
python main.py
```

# Platforms Supported
- [x] Windows
- [x] Linux
- [x] Android (termux)
- [x] Macos
